# docs/

📖 **The documentation lives at [react-native-nitro-thumbnail.vercel.app](https://react-native-nitro-thumbnail.vercel.app)** —
a single source of truth, authored as MDX in [`../website/pages`](../website/pages) and
deployed from there.

This folder only holds **media assets** referenced by the root README and the docs site:

- `assets/architecture.png` / `.svg` — the "one API, three engines" diagram
- `assets/demo-thumbnail.jpg` — a real thumbnail produced by `createThumbnail()`
- `assets/demo-ios.mp4` / `demo-ios-poster.jpg` — the iOS example-app demo
- `assets/CREDITS.md` — attribution for the demo clips (Sintel, CC-BY)

To edit the docs, change the matching page under `../website/pages/` and push — Vercel
redeploys automatically.
